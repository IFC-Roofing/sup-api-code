# Building Code Upgrades — Items Carriers Miss

## Why This Matters
Insurance must restore to current building code, not just pre-loss condition. Many adjusters write estimates based on what was there before, missing code upgrades that are now required by IRC or local amendments.

## Common Code Upgrades (Texas / DFW)

### Ice & Water Shield
- **IRC R905.1.2** — Ice barrier required at eaves in areas where average January temp ≤ 25°F
- **Texas application:** Not universally required by state code, but many local amendments require it
- **Practice:** Most DFW jurisdictions require I&W at valleys and eaves minimum
- **Qty:** Eaves LF + Valleys LF × 3' width
- **When to add:** If carrier doesn't include it and local code requires it

### Drip Edge
- **IRC R905.2.8.5** — Drip edge required at eaves and rakes of shingle roofs
- **Always required** for new roof installation in Texas
- **Qty:** Full eaves + rakes LF from EagleView
- **Common carrier miss:** They include drip edge but at lower quantity than actual perimeter

### Starter Strip
- **Manufacturer requirement** — Every shingle manufacturer requires starter at eaves and rakes
- **Not technically IRC** but manufacturer installation guidelines = industry standard
- **Carriers who bundle into waste (USAA)** are the outlier — most pay separately

### Ventilation
- **IRC R806.1** — Minimum 1 SF net free ventilation per 150 SF of attic floor
- **If existing ventilation doesn't meet code** → must upgrade during re-roof
- **Common items:** Ridge vent (if not present), additional intake vents, power vent replacement

### Steep Roof Safety
- **OSHA 1926.501(b)(11)** — Fall protection required at 6' above lower level
- **Not a line item** but supports steep charge justification
- **Carriers who deny steep charges** can be rebutted with safety/labor requirements

### Underlayment
- **IRC R905.1.1** — Underlayment required under shingles
- **30lb vs 15lb:** Premium shingles may require heavier underlayment per manufacturer specs
- **Synthetic underlayment:** Increasingly required/recommended for higher-grade shingles

## When to Use Code References in F9s
- DO reference IRC sections when the code clearly requires the item
- DO NOT fabricate code citations — only use what you can verify
- DO reference "manufacturer installation guidelines" when specific code section isn't available
- DO reference "industry standard practice" as fallback (weakest but still valid)

## Code Reference Format in F9s
"Per IRC Section R905.2.8.5, drip edge is required at eaves and rakes for asphalt shingle roof installations."

NOT: "Per your policy, drip edge is required..." (forbidden — never cite policy)

## Local Amendments
- DFW jurisdictions may have stricter requirements than state IRC
- When uncertain about local code: note in analysis, recommend verification before submitting
- City of Fort Worth, Dallas, Arlington, etc. each have their own amendments
