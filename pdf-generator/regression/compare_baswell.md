## ROOT CAUSE ANALYSIS

### CRITICAL ISSUES (leaving money on the table)

1. **R&R vs Install-only** — This is the #1 problem. Hip/ridge, drip edge, pipe jack, exhaust cap, turbine vent, chimney chase — Sup used INSTALL-ONLY descriptions (no remove component). Human used R&R for all of them. R&R adds a remove rate per unit = significantly more money. On drip edge alone that's $342 lost. Across all items it's probably $700-900.

2. **Missing paint items** — Sup doesn't add "Prime & paint" companion items for pipe jacks or roof vents. Human adds them because the existing ones are painted, so restoration requires priming and painting the new ones. That's $410 missed.

3. **Missing chimney flashing** — Sup has chimney chase cover but NOT chimney flashing. The rule in MEMORY.md says "chimney bid = cap/chase ONLY, always add separate Xactimate chimney flashing." Sup's own rule, not followed. $641 missed.

4. **Missing meter mast** — $848. This is a judgment call (depends on property), but human included it.

5. **Missing apron/counter flashing** — Sup has step flashing but NOT apron flashing. $189 missed.

6. **Wrong bid scope** — Sup used Grizzly arbor ($845), human used Grizzly porch paint ($650). Need to verify which is correct for this project.

### MODERATE ISSUES

7. **Pricelist version** — FEB26 vs MAR26. This creates $200-300 total difference across all items. Not a bug per se, but the latest pricelist should always be used.

8. **Tear-off remove component** — Human has a remove rate on tear-off ($66.73), Sup has $0. Tear-off IS removal — it should have a remove component.

### MINOR ISSUES

9. **Right elevation window screen** — Human has 1 EA separately. Sup missed it. $79.
10. **Section naming** — Sup uses "Dwelling Roof", human uses "Roof". Both work but inconsistent.
11. **Rain diverter placement** — Sup puts it in roof section, human puts it in gutters. Both valid.
